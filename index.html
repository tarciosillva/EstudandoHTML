Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso por que a linguagem HTML é
responsável apenas pela marcaação dos elementos na janela do navegador. O HTML não cria nada, ele manda buscar ou ultiliza as
letrinhas que você digita.

 Parágrafos

Para colocarmos parágrafos, usamos os lementos "p" e "/p". Para fazer uma quebra de linha, mantendo o mesmo parágrafo,
usamos o elemento "br". Temos ainda parágrafos especiais, denominados títulos, cabeçalhos ou headings em que são usados
elementos de "h1" a "h6" e seus respectivos fechamentos de "h1" a "h6".

 Imagens

Uma imagem aparece na janela do navegador pelo uso do elemto "img". Esse elemento possui o atributo "src", responsável pela
localização da imagem. Para trabalhar melhor com a imagem, você pode colocá-la dentro das tags de marcação de parágrafo.
